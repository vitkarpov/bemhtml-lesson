## Декларативный шаблонизатор: основы

Почитать — https://ru.bem.info/technology/bemhtml/v2/rationale/

**Отличия декларативного от императивного шаблонизатора**

### Входные данные

```
{
  items: [
    { text: '1' },
    { text: '2' }
  ]
}
```

### Шаблоны

Императивный

```html
<ul class="menu">
    [% foreach item in items %]
        <li class="menu__item">
            [% item.text %]
        </li>
    [% end %]
</ul>
```

Декларативный

```js
block('menu')(
    tag()('ul'),
    elem('item').tag()('li')
)
```
